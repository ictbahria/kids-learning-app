# Kids Learning App

A mobile application designed to teach children alphabets and numbers through interactive tracing activities.

## Features

- 🔤 Alphabet Tracing - Learn letters A-Z with guided tracing
- 🔢 Number Tracing - Learn numbers 0-9 with interactive practice
- 🎨 Colorful UI - Engaging and kid-friendly interface
- 📱 Cross-platform - Works on both iOS and Android
- 🎵 Audio Feedback - Sound effects for learning
- 🏆 Progress Tracking - Track learning progress

## Tech Stack

- React Native
- React Navigation
- SVG for drawing functionality
- Lottie animations

## Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn
- React Native CLI
- Xcode (for iOS development)
- Android Studio (for Android development)

### Installation

```bash
# Clone the repository
git clone https://github.com/ictbahria/kids-learning-app.git
cd kids-learning-app

# Install dependencies
npm install

# For iOS
npm run ios

# For Android
npm run android
```
